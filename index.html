List of assumptions based on understanding of the case study:

  • Investment has to start with a Lump Sum if Lump Sum Amount has been chosen
  • Lump Sum amount can only be a maximum of R30,000
  • Lump Sum amount can only be made once. Assumption that we cannot put more lump sum amount even for the next tax year
  • Debit Order Month has to start at least one month after Lump Sum month if opted
  • Debit Order can be optional but once started, continues indefinitely (i.e consider following tax year)
  • Debit Order amount can only be a maximum of R2500 irrespective of when it is starting throughout the year because over the following tax year, that amount will continue to occur per month, thus R2500 is the maximum to go indefinitely in order to never go above the threshold of R30000 for total yearly contributions
  • Given that drop downs should only contain months from January to December, the page has thus been built only for 1 tax year
  • Due to request for a simple page, this was built using JavaScript, HTML, CSS and Bootstrap, instead of preferred tech stack which is Reactjs
